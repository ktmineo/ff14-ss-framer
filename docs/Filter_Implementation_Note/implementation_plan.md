# 実装計画

## 概要
Note記事「LUT｜MIST」で言及されているような「映画のワンシーン」のような雰囲気（Mist/Cinematic）を再現する新しいフィルター「Mist」を実装します。
既存の4種類のフィルターとは異なるアプローチ（S字カーブ、フェード効果）を用い、差別化を図ります。

## 変更内容

### `js/filters/lut-presets.js`
- **[MODIFY]** `lut-presets.js`
    - 新しいヘルパー関数 `applySCurve` (S字トーンカーブ) を追加。
    - 新しいヘルパー関数 `applyFade` (黒浮き効果) を追加。
    - ミスト風の変換ロジック `mistTransform` を実装。
        - わずかなフェード（黒をグレーに持ち上げ）
        - S字カーブによるシネマティックなコントラスト
        - 彩度を僅かに下げる
    - プリセット定義に `'mist'` を追加。

### `js/app.js`
- **[MODIFY]** `app.js`
    - `filterPresets` に `mist` を追加。
    - 必要に応じてノイズ設定などを調整（Mistなので少しノイズがあっても良いかもしれないが、まずはLUTで質感を出す）。

## 検証計画
### 手動検証
- 画像をアップロードし、Mistフィルターを適用。
- 以下の特徴が表れているか確認：
    - 黒が真っ黒ではなく、少し浮いている（マットな質感）。
    - 全体的に映画のような落ち着いたトーン。
    - 既存のAnalog（単なるウォーム系）やNoir（白黒）と明確に違うこと。
